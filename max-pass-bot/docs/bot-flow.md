# Траектория кнопок (30–60 сек)

1. `/start` или первый запуск бота → дисклеймер → «Принимаю»
2. Главное меню → «Новая заявка»
3. Зона (4 кнопки) → Время (утро/день/вечер) → Дата (сегодня/завтра/послезавтра)
4. ФИО (текст) → Цель визита (текст)
5. Подтверждение → «Отправить» → номер `GP-YYYY-NNNNNN`, статус `pending`

Администратор ИБ: «Очередь ИБ» → карточка → Одобрить / Отклонить / Уточнение / Закрыть.
